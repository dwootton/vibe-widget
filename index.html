Vibe Widget

Generate interactive visualizations from natural language descriptions using large language models.

Vibe Widget is a Python library that transforms natural language prompts into production-ready interactive visualizations. By combining language model inference with programmatic rendering, it eliminates the need for manual charting library configuration while maintaining full flexibility over the output.

The system accepts pandas DataFrames and textual descriptions, then synthesizes React-based visualization components that can be embedded in notebooks, exported as standalone HTML, or integrated into applications.

Walkthrough

1Installation

Install via pip or uv package manager:

pip install vibe_widget

2Configure API Access

Vibe Widget uses the Anthropic API for component generation. Set your API key as an environment variable:

export ANTHROPIC_API_KEY='your-api-key-here'

Alternatively, pass the key directly to the create function via the api_key parameter:

vw.create(
    "an interactive scatterplot",
    df,
    api_key="your-api-key-here"
)

3Prepare Your Data

Create or load a pandas DataFrame containing the data to visualize:

import pandas as pd

df = pd.DataFrame({
    'height': [150, 160, 170, 180, 190],
    'weight': [50, 60, 70, 80, 90]
})

4Generate the Visualization

Call vibe_widget.create with a natural language description and your DataFrame:

import vibe_widget as vw

vw.create("an interactive scatterplot of height and weight", df)

The function returns an HTML file containing a self-contained interactive visualization. The process involves three stages:

  1. Data structure analysis and schema extraction
  2. React component synthesis via Claude API
  3. Bundle compilation and HTML generation

5Export and Deploy

Specify an output path to save the visualization as a file:

vw.create(
    "an interactive line chart showing trends over time",
    df,
    output_path="output/visualization.html"
)

The resulting HTML file is fully standalone and can be opened directly in a browser, embedded in documentation, or served via static hosting.

Capabilities

Natural Language Interface

Describe visualizations in plain language rather than configuring chart libraries manually. The system interprets semantic intent and maps it to appropriate visual encodings.

React-Based Rendering

Generated components use modern React patterns, enabling rich interactivity and state management. The output integrates seamlessly with existing React applications or functions independently.

Pandas Integration

Direct DataFrame support allows for immediate visualization of tabular data structures commonly used in data analysis workflows.

Extensible Architecture

The modular design accommodates custom LLM providers and rendering strategies, allowing adaptation to specific use cases or deployment constraints.

Note: Vibe Widget generates code via API inference. Review generated components before deploying to production environments to ensure they meet security and performance requirements.
